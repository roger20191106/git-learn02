Git 常用指令
- 初始化数据库：git init
- 查询当前状况：git status
- 将档案加入到索引： git add .
- 将索引档案变成一个更新（commit）：git commit -m “修改内容”
- 观察commit历史记录： git log
下载远端数据库： git clone 数据库网址
更新远端数据库： git push origin master


Git的架构
工作目录 <-->  索引 （staging area） <---> 本地数据库 localrepository <---> 远端数据库

git status 检查工作目录现在的状况（新增、修改、删除）
